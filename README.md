# 🚀 Landing Page CSS

Uma landing page moderna, responsiva e estilizada com CSS puro. Este projeto demonstra como criar uma interface visualmente impactante com recursos avançados de estilização, mantendo simplicidade e performance.

## ✨ Destaques da Página

- Design escuro com contraste elegante
- Gradientes animados aplicados ao texto
- Cartões interativos com hover e sombra
- Layout totalmente responsivo usando Grid e Flexbox
- Tipografia moderna e espaçamento fluido

---

## 💡 Tecnologias Utilizadas

- **HTML5**
- **CSS3**

---

## 🧠 Funcionalidades CSS Principais

- `box-sizing: border-box` – Controle total do layout
- `background: linear-gradient(...)` – Gradientes aplicados em botões e textos
- `background-clip: text` + `-webkit-text-fill-color: transparent` – Texto com cor de gradiente
- `max-width`, `margin: 0 auto`, `padding` – Layout centralizado e fluido
- `display: grid`, `grid-template-columns: repeat(...)` – Layout de benefícios responsivo
- `display: flex`, `justify-content: center` – Organização de elementos em seções
- `transition`, `transform: scale(...)` – Animações suaves ao interagir
- `position: relative`, `::before`, `:hover` – Cartões com efeitos visuais dinâmicos
